# SportAnalytics Frontend

## Setup

1. Stelle sicher, dass Node.js installiert ist.
2. Installiere die Angular CLI global (falls noch nicht vorhanden):
   ```
   npm install -g @angular/cli
   ```
3. Installiere die Abhängigkeiten:
   ```
   npm install
   ```
4. Starte das Frontend:
   ```
   npm start
   ```

Das Frontend läuft standardmäßig auf http://localhost:4200
